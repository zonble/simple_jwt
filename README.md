# simple_jwt

A simple library helps to generate [JSON Web Tokens](https://jwt.io/).

The library supports HS256 algorithm only.

## Usage

A simple usage example:

    import 'package:simple_jwt/simple_jwt.dart';

    main() {
      var awesome = new Awesome();
    }
